# GSPDT Corporate Construction Website

Welcome to the repository for **GSPDT Corporate Construction Website**, a project showcasing the company profile and portfolio of PT. Global Solution Provider in Bandung. This website highlights our expertise, projects, and services in the construction industry.

## About the Project

This website is designed to present the professional image and extensive portfolio of PT. Global Solution Provider. Our goal is to provide potential clients with a comprehensive view of our capabilities and achievements.

### Features

- **Company Profile**: Detailed information about our company's history, mission, and values.
- **Project Portfolio**: Showcases our completed and ongoing construction projects with images and descriptions.
- **Services Overview**: Information on the various construction and project management services we offer.
- **Contact Information**: Easy access to our contact details for inquiries and business opportunities.

## Technology Stack

The website is built using modern web technologies to ensure a responsive and engaging user experience.

### Key Technologies Used:

- **HTML/CSS**: For structuring and styling the web pages.
- **JavaScript**: For interactive features and dynamic content.
- **Bootstrap**: For responsive design and layout.
- **PHP**: For server-side scripting and handling form submissions.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Web Server (e.g., Apache, Nginx)
- PHP: [Download and Install PHP](https://www.php.net/downloads)
- Git: [Download and Install Git](https://git-scm.com/downloads)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/25chris/gspdt.co.id.git
   ```
2. Navigate to the project directory
   ```sh
   cd gspdt.co.id
   ```
3. Move the files to your web server's root directory (e.g., `htdocs` for XAMPP)
4. Start your web server and navigate to `http://localhost` to view the website
